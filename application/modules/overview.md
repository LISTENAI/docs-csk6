# 概述
Zephyr系统提供了丰富的组件，如LVGL、日志组件等，另外CSK6 SDK还增加了一些功能组件例如WiFi manager、音频播放、录音等系统组件，应用开发可以通用引用系统组件的方式获取组件模块的功能，复用性强，提高了应用开发的便捷性。
